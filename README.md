# Knowing It When We Know It

A short live talk by Anna — a reader's tour through Jennifer Nagel's *Knowledge: A Very Short Introduction*, from "what even is knowledge?" through Gettier, scepticism and testimony, landing on why contextualism is the most convincing view.

Built with [Slidev](https://sli.dev). The deck lives in [`slides.md`](./slides.md); the pink-and-cutesy theme overrides are in [`style.css`](./style.css).

## Quick start

```bash
npm install        # or: make install
npm run dev        # or: make live  — opens http://localhost:3030
```

Edit `slides.md` while the dev server runs and the deck hot-reloads.

## Handy commands

| Command          | What it does                                  |
| ---------------- | --------------------------------------------- |
| `make live`      | Run the dev server and open it in the browser |
| `make pdf`       | Export the deck to `knowledge-vsi-summary.pdf`|
| `make build`     | Build a static site into `dist/`              |
| `make stop`      | Free the dev-server port (3030)               |
| `make help`      | List all targets                              |

(Each has an `npm run` equivalent — see [`package.json`](./package.json).)
